# OVERVIEW
This folder contains R code required to run a sample size calculator
for a binary outcome setting.

# HOW TO USE
Run ss_calculator_sample_code.R for an example of using the sample size calculator


# FILE DESCRIPTIONS
	1. ss_calculator_sample_code.R
	Example R script demonstrating how to use the sample size
	calculator for a binary outcome.

	2. toy_data.RDS
	Toy dataset containing a binary outcome with three treatment
	arms under equal probability assignment.

	3. dgm.R
	R script used to generate the toy dataset stored in toy_data.RDS.


	4. ss_calc.R
	Core sample size calculator implementation for a binary outcome.

	5. wcls_cat_binary_outcome.R
	Estimator for the marginal excursion effect using weighted and centered least squares (WCLS).
